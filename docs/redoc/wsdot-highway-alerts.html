Wsdot Highway Alerts API - Washington State Department of Transportation APIs

highwayAlerts

Each Alert item represents real-time traffic incidents, road conditions, construction, and other events affecting Washington State highways. These include location details, impact levels, start/end times, and estimated duration.

Returns a single Alert object for specified AlertID.

path Parameters
AlertID
required
number (AlertByIdInputAlertIDParam)

Unique highway alert identifier, as an integer ID. E.g., '669625' for I-405 construction alert, '468632' for SR-520 flammable cargo restriction, '669582' for I-405 collision alert. Used to retrieve specific alert details.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/highwayalerts/highwayalertsrest.svc/getAlertAsJson?AlertID={AlertID}

Request samples

import { fetchDottie, getAlertById } from 'ws-dottie';

const data = await fetchDottie({
  endpoint: getAlertById,
  params: {
    "AlertID": 468632
  },
  fetchMode: 'native',
  validate: true
});

console.log(data);

Returns an array of Alert objects matching specified search criteria.

path Parameters
StateRoute
required
string (SearchAlertsInputStateRouteParam)

State route identifier formatted as three-digit number, as a route identifier. E.g., '005' for I-5, '405' for I-405, '090' for I-90. Used to filter alerts by specific highway route.

Region
required
integer (SearchAlertsInputRegionParam)

WSDOT region identifier code, as an integer. Valid values: 7 (Eastern), 8 (North Central), 9 (Northwest), 10 (Olympic), 11 (South Central), 12 (Southwest). E.g., '9' for Northwest region, '7' for Eastern region. Used to filter alerts by WSDOT region.

SearchTimeStart
required
string (SearchAlertsInputSearchTimeStartParam)

Start time for alert search range, in ISO datetime format. E.g., '2025-11-01T00:00:00Z' for November 1, 2025 midnight UTC. Used to filter alerts occurring after this time.

SearchTimeEnd
required
string (SearchAlertsInputSearchTimeEndParam)

End time for alert search range, in ISO datetime format. E.g., '2025-11-02T23:59:59Z' for November 2, 2025 end of day UTC. Used to filter alerts occurring before this time.

StartingMilepost
required
number (SearchAlertsInputStartingMilepostParam)

Starting milepost value for milepost range filtering, as a decimal. E.g., '20.3' for milepost 20.3, '11.5' for milepost 11.5. Used with EndingMilepost to filter alerts within milepost range.

EndingMilepost
required
number (SearchAlertsInputEndingMilepostParam)

Ending milepost value for milepost range filtering, as a decimal. E.g., '22.61' for milepost 22.61, '25.91' for milepost 25.91. Used with StartingMilepost to filter alerts within milepost range.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/highwayalerts/highwayalertsrest.svc/searchAlertsAsJson?StateRoute={StateRoute}&Region={Region}&SearchTimeStart={SearchTimeStart}&SearchTimeEnd={SearchTimeEnd}&StartingMilepost={StartingMilepost}&EndingMilepost={EndingMilepost}
