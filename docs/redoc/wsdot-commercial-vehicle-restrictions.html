Wsdot Commercial Vehicle Restrictions API - Washington State Department of Transportation APIs

cv-restriction-data

Each CVRestrictionData item represents current commercial vehicle restrictions for Washington State highways, including weight limits, height clearances, axle restrictions, and location details. These items provide essential regulatory information for bridge and road restrictions needed for commercial vehicle routing and permit processing.

Returns an array of CVRestrictionData objects containing restriction information for all Washington State highways.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/cvrestrictions/cvrestrictionsrest.svc/getCommercialVehicleRestrictionsAsJson

Request samples

  • JavaScript
import { fetchDottie } from 'ws-dottie';
import { fetchCommercialVehicleRestrictions } from 'ws-dottie/wsdot-commercial-vehicle-restrictions/core';

const data = await fetchDottie({
  endpoint: fetchCommercialVehicleRestrictions,
  fetchMode: 'native',
  validate: true
});

console.log(data);

cv-restriction-data-with-id

Each CVRestrictionDataWithId item represents current commercial vehicle restrictions for Washington State highways with unique tracking identifiers. These include weight limits, height clearances, axle restrictions, and restriction details for both bridge and road restrictions across the statewide coverage area.

Returns an array of CVRestrictionDataWithId objects containing restriction information with unique identifiers for all Washington State highways.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/cvrestrictions/cvrestrictionsrest.svc/getCommercialVehicleRestrictionsWithIdAsJson
