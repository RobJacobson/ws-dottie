Wsdot Toll Rates API - Washington State Department of Transportation APIs

toll-rates

Each TollRate item represents current toll pricing information for high occupancy toll lanes across Washington state. Each rate contains dynamic pricing data, trip location details, and real-time update timestamps for congestion management.

Returns multiple TollRate items for all high occupancy toll lanes statewide.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTollRatesAsJson

Request samples

  • JavaScript
import { fetchDottie, getTollRates } from 'ws-dottie';

const data = await fetchDottie({
  endpoint: getTollRates,
  fetchMode: 'native',
  validate: true
});

console.log(data);

toll-trip-rates

Each TollTripRates item represents comprehensive toll rate data for specific trips including pricing details, informational messages, and version tracking. Each container supports current data retrieval, historical date ranges, and version-specific queries.

Returns single TollTripRates item for specific version.

path Parameters
Version
required
number (TripRatesByVersionInputVersionParam)

Version number for toll trip rates query, as an integer. E.g., '371529' for version 371529. Used to retrieve specific version of toll trip rates data.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTripRatesByVersionAsJson?Version={Version}

Request samples

  • JavaScript
import { fetchDottie, getTripRatesByVersion } from 'ws-dottie';

const data = await fetchDottie({
  endpoint: getTripRatesByVersion,
  params: {
    "Version": 352417
  },
  fetchMode: 'native',
  validate: true
});

console.log(data);

Returns multiple TollTripRates items for specified date range.

path Parameters
FromDate
required
string (TripRatesByDateInputFromDateParam)

Start date for historical trip rates query, as a date string. E.g., '2025-11-01' for November 1, 2025. Used to define beginning of date range for historical toll rate queries.

ToDate
required
string (TripRatesByDateInputToDateParam)

End date for historical trip rates query, as a date string. E.g., '2025-11-02' for November 2, 2025. Used to define end of date range for historical toll rate queries.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTripRatesByDateAsJson?FromDate={FromDate}&ToDate={ToDate}

import { fetchDottie, getTripRatesByDate } from 'ws-dottie';

toll-trip-version

TollTripVersion provides version and timestamp information for toll trip data, enabling cache management and data freshness tracking.

Returns current version and timestamp information for toll trip data.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTollTripVersionAsJson

Request samples

  • JavaScript
import { fetchDottie, getTollTripVersion } from 'ws-dottie';

const data = await fetchDottie({
  endpoint: getTollTripVersion,
  fetchMode: 'native',
  validate: true
});

console.log(data);
