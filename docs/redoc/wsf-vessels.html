cache-flush-date-vessels

Represents the timestamp of when any static endpoint data for the wsf-vessels API was last updated. This information helps applications determine when to refresh cached vessels information through cache invalidation.

Returns the timestamp of when any static endpoint data for the wsf-vessels API was last updated.

Responses

Production server

https://www.wsdot.wa.gov/ferries/api/vessels/rest/cacheflushdate

Request samples

  • JavaScript
import { fetchDottie } from 'ws-dottie';
import { fetchCacheFlushDateVessels } from 'ws-dottie/wsf-vessels/core';

const data = await fetchDottie({
  endpoint: fetchCacheFlushDateVessels,
  fetchMode: 'native',
  validate: true
});

console.log(data);

Returns multiple VesselHistory objects for the specified vessel and date range.

path Parameters
VesselName
required
string (FetchVesselHistoriesByVesselNameAndDateRangeInputVesselNameParam)

Vessel name for history query, as a human-readable description. E.g., 'Tacoma' for vessel Tacoma, 'Chelan' for vessel Chelan. Required to filter history records for a specific vessel.

DateStart
required
string (FetchVesselHistoriesByVesselNameAndDateRangeInputDateStartParam)

Start date for history query date range, as a date string. E.g., '2025-09-01' for September 1, 2025. Defines the beginning of the date range for retrieving historical vessel departure and arrival records.

DateEnd
required
string (FetchVesselHistoriesByVesselNameAndDateRangeInputDateEndParam)

End date for history query date range, as a date string. E.g., '2025-09-30' for September 30, 2025. Defines the end of the date range for retrieving historical vessel departure and arrival records.

Responses

Production server

Request samples

  • JavaScript
import { fetchDottie } from 'ws-dottie';
import { fetchVesselsVerboseByVesselId } from 'ws-dottie/wsf-vessels/core';

const data = await fetchDottie({
  endpoint: fetchVesselsVerboseByVesselId,
  params: {
    "VesselID": 68
  },
  fetchMode: 'native',
  validate: true
});

console.log(data);
