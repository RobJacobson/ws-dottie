toll-rates

Current toll rates for high occupancy vehicle (HOV) toll lanes statewide.

List current toll rates for all HOV toll lanes statewide.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTollRatesAsJson

Request samples

  • JavaScript
import { fetchTollRates } from 'ws-dottie/wsdot-toll-rates/core';

const data = await fetchTollRates({
  fetchMode: 'native',
  validate: true
});

console.log(data);

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/tollrates/tollratesrest.svc/getTripRatesByDateAsJson?FromDate={FromDate}&ToDate={ToDate}

  • JavaScript
import { fetchTripRatesByDate } from 'ws-dottie/wsdot-toll-rates/core';

  • JavaScript
