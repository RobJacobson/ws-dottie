Search highway alerts by route, region, time range, and milepost.

path Parameters
StateRoute
required
string (SearchAlertsInputStateRouteParam)

State route identifier as three-digit number, or undefined to include all routes.

Region
required
integer (SearchAlertsInputRegionParam)

WSDOT region identifier code: 7 = Eastern, 8 = North Central, 9 = Northwest, 10 = Olympic, 11 = South Central, 12 = Southwest, or undefined to include all regions.

SearchTimeStart
required
string (SearchAlertsInputSearchTimeStartParam)

UTC datetime when the search time range begins, in ISO format, or undefined for no start limit.

SearchTimeEnd
required
string (SearchAlertsInputSearchTimeEndParam)

UTC datetime when the search time range ends, in ISO format, or undefined for no end limit.

StartingMilepost
required
number (SearchAlertsInputStartingMilepostParam)

Starting milepost value for range filtering, or undefined for no start limit.

EndingMilepost
required
number (SearchAlertsInputEndingMilepostParam)

Ending milepost value for range filtering, or undefined for no end limit.

Responses

Production server

https://www.wsdot.wa.gov/traffic/api/highwayalerts/highwayalertsrest.svc/searchAlertsAsJson?StateRoute={StateRoute}&Region={Region}&SearchTimeStart={SearchTimeStart}&SearchTimeEnd={SearchTimeEnd}&StartingMilepost={StartingMilepost}&EndingMilepost={EndingMilepost}
